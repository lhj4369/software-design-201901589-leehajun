## 기능 요구사항

### 1. 학생 성적 관리
- 학기별 성적 기록·수정
- 입력 후 **총점·평균·등급** 자동 계산·표시
- 학생별·과목별 성적 조회
- **전 교과목 성적 레이더 차트** 시각화

### 2. 학생부 관리
- 기본 정보(이름, 학년, 반, 번호 등) 및 성적 외 정보(출결, 특기사항 등) 기록
- 학생부 항목 **교사 수정·추가**

### 3. 피드백 제공
- 학생별 피드백 작성·저장
- 성적 외 **행동·출결·태도** 등 다양한 항목
- 학생·학부모 **공개 여부** 옵션

### 4. 상담 내역 관리
- 상담 기록: 날짜, 주요 내용, 다음 상담 계획
- **다른 교사와 상담 내역 공유**
- 교사 간 상담 **검색·필터**

### 5. 학생 정보 검색·조회
- 성적·상담·피드백 통합 검색
- **기간·과목** 등 필터

### 6. 알림
- 성적 입력, 피드백, 상담 업데이트 등 알림
- 학부모·학생에게 성적·피드백 갱신 시 **자동 알림**

## 사용자 시나리오

1. **교사 A** — 성적 입력 후 해당 학생에게 피드백 작성·저장  
2. **교사 B** — 학생 상담 내역 조회, 이전 상담 확인 후 후속 상담 계획  
3. **학부모** — 자녀 성적 조회, 교사 피드백 확인
